# 随机身份 — MVP Spec

## 它做什么

单页体验：同一道选择，先走女线再看男线——让用户自己发现反馈不一样。

## 给谁用

校内同龄人和社交媒体上的普通公众——没玩过这类互动作品、愿意花 2–3 分钟试一次的人。

## 核心流程

1. 点「开始」——看到一行说明：接下来会用两种身份各经历一次相同选择。
2. 读场景（公共演讲前：化妆 vs 练稿），选 `1` 或 `2`——收到女线 NPC 反馈（不标注性别）。
3. 点「继续」——同一道题、同一选项，收到男线反馈。
4. 看对照页——「你选了 ___，女线收到 ___，男线收到 ___。同样的选择，不同的反馈。」
5. 答 1 道玩后题（可选写 1 句）——结束。

## 不在 MVP 里

- 规规向导、穿越设定、职业选择（歌手 / CEO）
- 隐藏拒绝（菜单外输入「拒绝」「凭什么」）
- 行动值分配、隐藏达标线
- 能力 / 压力 / 事业进度数值、压力强制休息线
- 5 轮剧本、多个世界日、双线多段穿插
- 玩前问卷、CSV 日志、自动统计
- 英文页、精美 UI、论文级对照组

## 怎么收集反馈

- **页内必做**：玩后单选题——「两次结果不一样，你的第一反应更接近？」（奇怪/不公平 · 正常 · 说不清）
- **页内选做**：开放一句——「这让你想到生活中的什么？」（可跳过）
- **线下必做**：给 5 个朋友口头问 3 句——哪步最别扭？第几步发现不一样？像不像真的？
- **分发**：校内 5–10 人 + 社交媒体一条链接；开头注明匿名、可退出、不是心理测试
