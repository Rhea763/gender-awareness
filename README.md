# Random Identity / 随机身份

How can people recognize and critically examine the gender expectations that shape everyday life?

**Random Identity** is a short browser-based interactive experience about everyday gender expectations. The core interaction is simple: the user makes the same choice in two identity paths, then compares how the feedback changes.

## Live Site

https://gender-awareness.vercel.app/

## What Is In This Repository

- `index.html` — the main Vercel entry page, with the short MVP experience and research summary.
- `game.html` — a fuller traversal demo with more game systems.
- `spec.md` — the MVP plan and research question.
- `midterm-draft.md` — the English midterm presentation draft.
- `research/` — notes, sources, data excerpts, archived prototypes, and research materials.

## How To Open Locally

Double-click `index.html`, or open the folder in VS Code and use Live Server.

No build step is required for the website.
